# Claude Dev

Tools and guidance for extending Claude Code.

## Skills

### skill-authoring

Guidance for creating effective skills following:

- [Agent Skills Specification](https://agentskills.io) - The open format spec
- [Skill Authoring Best Practices](https://platform.claude.com/docs/en/agents-and-tools/agent-skills/best-practices)

**Core Guidance (SKILL.md)**

- Skill structure and format
- Required/optional frontmatter fields
- Core principles (concise, degrees of freedom, progressive disclosure)
- Writing effective descriptions
- Workflow patterns
- Anti-patterns to avoid
